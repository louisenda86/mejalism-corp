# Mejalism Corp Rev 2

Rev 2 cleans up the original build into a more professional interactive art-direction system.

## Changes

- Refined the blue-and-white reference style with slimmer borders, calmer page texture, and tighter type scale.
- Reduced the p5.js background density so the content panels read more clearly.
- Added an animated globe element to the New York Garage subheader.
- Added a visible `REV 2` badge in the status console and footer.
- Preserved the interactive cursor, pretext text animation, click sound, ambient sound, dark/light mode, and idle LED behavior.

## Run

```sh
python3 -m http.server 5177
```

Open `http://localhost:5177`.
